# 🎓  ID Card Generator - Function & Events

A professional, modern, and high-quality ID card generation system designed for college functions and events. This tool helps organizers quickly generate, print, and download participant ID cards with unique identification and QR codes.

## ✨ Key Features
- **Modern UI/UX**: Premium design using glassmorphism and the "Outfit" font.
- **Form-Based Input**: Collects student Name, Department, Register Number, Year, Address, Mobile Number, and Blood Group.
- **Dynamic Unique ID**: Automatically generates a unique alphanumeric identifier for each participant.
- **Automatic QR Code Generation**: Encodes student details (Unique ID, Name, Dept, Reg No) into a scan-ready QR code.
- **Professional Layout**: Optimized ID card design for visual appeal and functional use.
- **Export Options**: 
    - 🖨️ **Print Ready**: Optimized print styles for standard browser printing.
    - 📄 **PDF Download**: High-quality PDF export supported by `html2pdf.js`.

## 🚀 How to Run the Project
This is a standard frontend project that runs entirely in the browser. No complex backend installation is required!

### Option 1: Standard Method
1.  **Extract/Navigate** to the project folder.
2.  **Open** the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge).
3.  **Fill** in the student details in the form.
4.  **Click** on "Generate ID Card" to see the preview.

### Option 2: Live Server (Recommended)
If you are using **VS Code**:
1.  Install the **Live Server** extension.
2.  Right-click on `index.html` and select **"Open with Live Server"**.
3.  This provides a localized development environment and ensures all features work seamlessly.
